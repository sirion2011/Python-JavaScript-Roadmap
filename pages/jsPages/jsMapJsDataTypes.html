Типы данных в JavaScript

В JavaScript существует два основных типа данных: примитивные и ссылочные.

Примитивные типы:

  • string — строка
  • number — число
  • boolean — логическое значение
  • null — пустое значение
  • undefined — значение не задано
  • symbol — уникальный идентификатор
  • bigint — большие целые числа

Ссылочные типы:

  • object — объект
  • array — массив (частный случай объекта)
  • function — функция (также объект)

Проверка типа: оператор typeof

typeof "hello";           // "string"
    typeof 42;                // "number"
    typeof true;              // "boolean"
    typeof undefined;         // "undefined"
    typeof null;              // "object" (!) баг языка
    typeof Symbol("id");     // "symbol"
    typeof 10n;               // "bigint"
    typeof {a:1};             // "object"
    typeof [1, 2];            // "object"
    typeof function(){};      // "function"

NaN (Not a Number)

NaN — это специальное значение типа number, которое означает "не число".

typeof NaN;               // "number"
    NaN === NaN;             // false
    Number.isNaN(NaN);       // true

Явное и неявное приведение типов

Неявное:

'5' + 1;          // "51"
    '5' - 1;          // 4
    true + 1;         // 2
    null + 1;         // 1

Явное:

Number("42");     // 42
    String(100);      // "100"
    Boolean(0);       // false

null vs undefined

Оба значения означают отсутствие данных, но по-разному:

  • null — намеренное отсутствие значения
  • undefined — значение не установлено
let x;
    console.log(x);          // undefined
    
    let y = null;
    console.log(y);          // null

Передача по значению и по ссылке

Примитивы передаются по значению, объекты — по ссылке.


    let a = 5;
    let b = a;
    b = 10;
    console.log(a);          // 5
    
    let obj1 = { name: "Tom" };
    let obj2 = obj1;
    obj2.name = "Jerry";
    console.log(obj1.name);  // "Jerry"

Проверка массива и объекта

Array.isArray([1, 2, 3]);          // true
    typeof [1, 2, 3];                  // "object"
    typeof {a: 1};                     // "object"

Объектные обертки (object wrappers)

Когда вы вызываете методы у примитивов, JavaScript временно преобразует их в объект:

"hello".toUpperCase();    // "HELLO"
    // создается временный объект new String("hello")

Типы данных в JSON

JSON поддерживает только ограниченный набор типов:

  • string
  • number
  • boolean
  • null
  • array
  • object

JSON не поддерживает undefined, function, symbol

Сводная таблица типов и typeof

Тип             | typeof результат | Примитив
    ---------------|------------------|---------
    string          | "string"         | Да
    number          | "number"         | Да
    boolean         | "boolean"        | Да
    null            | "object" (баг)   | Да
    undefined       | "undefined"      | Да
    symbol          | "symbol"         | Да
    bigint          | "bigint"         | Да
    object          | "object"         | Нет
    array           | "object"         | Нет
