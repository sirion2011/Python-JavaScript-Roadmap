JavaScript Objects

Основные понятия JavaScript для начинающих

Что такое объект в JavaScript?

Объекты — это фундаментальные структуры в JavaScript, используемые для хранения коллекций значений и более сложных сущностей. Они состоят из пар "ключ: значение".

Создание объектов

  • Через литерал:
    • const person = {
            name: "Alice",
            age: 25,
            isStudent: false
        };
  • Через конструктор:
    • const person = new Object();
        person.name = "Bob";
        person.age = 30;

Доступ к значениям

Можно обращаться к свойствам через точку или скобки:

console.log(person.name);     // Alice
        console.log(person["age"]);   // 25

Изменение и добавление свойств

person.age = 26;
        person.city = "Minsk";

Удаление свойства

delete person.isStudent;

Вложенные объекты

Объекты могут содержать другие объекты:

const user = {
            name: "John",
            address: {
                city: "London",
                zip: "EC1A"
            }
        };
        
        console.log(user.address.city);

Методы объектов

Методы — это функции, находящиеся внутри объекта:

const user = {
            name: "Anna",
            greet() {
                console.log("Hello, " + this.name);
            }
        };
        
        user.greet();

this в объектах

Ключевое слово this ссылается на текущий объект:

const car = {
            brand: "Toyota",
            showBrand() {
                console.log(this.brand);
            }
        };

Object.keys, Object.values, Object.entries

Эти методы позволяют работать со свойствами объекта:

const obj = { a: 1, b: 2 };
        
        console.log(Object.keys(obj));   // ["a", "b"]
        console.log(Object.values(obj)); // [1, 2]
        console.log(Object.entries(obj)); // [["a", 1], ["b", 2]]

Копирование объектов

  • Поверхностное копирование: Object.assign(), spread оператор
    • const copy = { ...person };
  • Глубокое копирование: JSON-методы (подходит не всегда):
    • const deepCopy = JSON.parse(JSON.stringify(person));

Проверка свойств

"name" in person;          // true
        person.hasOwnProperty("age");  // true

Цикл for...in

Используется для перебора всех перечисляемых свойств объекта:

for (let key in person) {
            console.log(key + ": " + person[key]);
        }
